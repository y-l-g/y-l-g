# Youenn Le Gouedec

Laravel, Vue and Nuxt developer based in Brittany, France.

I build and maintain web applications for SaaS products and small teams: back-end architecture, front-end interfaces, billing, integrations, deployment workflows, and existing codebases that need to become easier to evolve.

## Focus

- Laravel and PHP applications with clear domain code, queues, webhooks, billing, APIs and admin workflows.
- Vue, Nuxt and Inertia front ends that stay readable as the product grows.
- SaaS foundations: accounts, teams, roles, subscriptions, emails, customer-facing dashboards and operational tooling.
- Production work with PostgreSQL, Redis, Docker, FrankenPHP, k3s, Argo CD, backups, logs and health checks.
- Code audits and refactoring when the goal is to reduce risk without blocking product delivery.

## Selected Work

- [Webvite](https://webvite.eu)  
  Production SaaS for small businesses. It handles customer websites, subscriptions, administration, custom domains, analytics and ongoing production maintenance.

- [Saasterkit](https://github.com/y-l-g/saasterkit)  
  Open-source Laravel, Vue and Inertia starter kit for SaaS products, with authentication, teams, roles, billing and admin features.

- [Notenn](https://notenn.com)  
  Web app for musicians to organize, edit, render and share scores written in ABC notation.

- [Pogo WebSocket](https://github.com/y-l-g/websocket)  
  Experimental Pusher-compatible WebSocket module for FrankenPHP and Laravel broadcasting.

- [Pogo](https://github.com/y-l-g/pogo)  
  Experimental request-scoped parallel job runtime for PHP on FrankenPHP.
